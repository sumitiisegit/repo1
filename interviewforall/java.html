Java Interview Questions:



1. What is Java?
Java is a programming language, which follows Object Oriented Design.



2. Which OOPs features does Java support?
Java suppports Abstraction, Encapsulation, Polymorphism.


3. Where is Java used?
Java is used as a server side programming language. It integrates well with many other technologies to help development of variety of applications.

4. What is JVM?
JVM is Java Virtual Machine. It provides an environment for development and runtime of Java applications.
